SERVICES

We provide some service on our nodes.

Linuxbrew

export PATH="/home/share/linuxbrew/bin:$PATH" , or you can add to '~/.profile'
Now, we set "tmux" in linuxbrew. You can update or install by "brew install" or "brew update"

Conda

For shortage of SSD, please don't install conda.
Add PATH="/home/share/miniconda3/bin:$PATH" to '~/.profile' and then you can use conda.

Vpn

You can visit google on our nodes by 'proxychains4'.
E.g. proxychains4 curl www.google.com It's supported by Pengyang Shao, we may update this function in a few months. You can visit here for detail.

Connecting to Internet

If network broken, input the following in shell:
1. export PATH="/home/user/login_Internet:$PATH"
2. login_net
